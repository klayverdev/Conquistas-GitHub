## 🧩 O que são as Conquistas do GitHub?

As **Achievements** (Conquistas) são emblemas exibidos no seu perfil do GitHub que reconhecem sua participação, contribuições e marcos na plataforma. Elas funcionam como troféus digitais e ajudam a demonstrar seu engajamento com a comunidade open source.

> 💡 **Você pode ocultar ou exibir suas conquistas** acessando: **Configurações → Perfil → Achievements**

## ✅ Como Ganhar Cada Conquista

### 🤝 Pair Extraordinaire
Faça um commit com co-autoria adicionando esta linha na mensagem do commit:
```
Co-authored-by: Nome <email@exemplo.com>
```
O pull request precisa ser mergeado para a conquista ser contabilizada.

---

### ⚡ Quickdraw
1. Abra uma issue ou pull request em qualquer repositório
2. Feche-a **em menos de 5 minutos**

> No caso de um PR, basta abrir e fechar antes que alguém revise.

---

### ⭐ Starstruck
Crie um repositório útil e relevante e divulgue para conseguir **16 estrelas**. Quanto mais útil o projeto (ferramentas, tutoriais, projetos open source), mais rápido você consegue.

> Possui níveis: 16 ⭐ → 128 ⭐ → 512 ⭐ → 4096 ⭐

---

### 🧠 Galaxy Brain
1. Encontre um repositório com a aba **Discussions** habilitada
2. Responda uma pergunta de forma precisa e completa
3. O dono do repositório precisa **marcar sua resposta como aceita**

> ⚠️ **Atenção:** Desde fevereiro de 2024, o GitHub removeu essa conquista das Discussions comunitárias para combater spam. Ainda é possível ganhar em Discussions de repositórios públicos normais.

---

### 🦈 Pull Shark
1. Crie um repositório (pode ser o seu próprio)
2. Faça um branch, edite algo e abra um pull request
3. Dê merge no PR — **repita pelo menos 2 vezes**

> Possui níveis: 2 PRs → 16 PRs → 128 PRs → 1024 PRs

---

### 🎲 YOLO
1. Tenha um repositório com outra conta como colaboradora
2. Abra um pull request
3. Dê **merge sem aguardar ou aprovar nenhuma revisão**

> A conquista é desbloqueada no momento do merge sem review aprovado.

---

### 💸 Public Sponsor
Acesse [github.com/sponsors](https://github.com/sponsors), escolha um projeto ou desenvolvedor e faça uma doação. O valor mínimo é **$5 USD** e a conquista aparece no seu perfil após a confirmação do pagamento.

## 🎯 Níveis das Conquistas (Tiers)

Algumas conquistas possuem níveis progressivos:

| Conquista | 🥉 Bronze | 🥈 Prata | 🥇 Ouro | 💎 Diamante |
|-----------|:---------:|:--------:|:-------:|:-----------:|
| Starstruck | 16 ⭐ | 128 ⭐ | 512 ⭐ | 4096 ⭐ |
| Pull Shark | 2 PRs | 16 PRs | 128 PRs | 1024 PRs |
| Pair Extraordinaire | 1 co-autoria | 10 | 24 | 48 |
| Galaxy Brain | 1 resposta | 8 | 16 | 32 |

## 💡 Dicas Gerais

- 🔍 **Visibilidade:** Suas conquistas aparecem no seu perfil público por padrão — você pode desativá-las em *Settings → Profile*
- 🎨 **Skin Tone:** Alguns emblemas variam de aparência com base na sua preferência de tom de pele em *Settings → Appearance*
- 📈 **Estratégia:** Comece pelas mais fáceis: **Quickdraw**, **YOLO** e **Pull Shark** são as mais simples de conquistar
- 🌍 **Engajamento real:** Contribuir com projetos open source de verdade acelera conquistas como **Open Sourcerer** e **Galaxy Brain**
